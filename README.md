# 日本語 — Japanese Learning Roadmap

A beautifully designed, interactive Japanese learning guide built in pure HTML, CSS, and JavaScript. Takes you from absolute zero to conversational fluency across 4 structured phases — no frameworks, no dependencies, one single file.

---

## 🌸 Live Demo

> Deploy via GitHub Pages and paste your link here.

---

## ✨ Features

- **4-phase roadmap** — structured progression from N5 beginner to N1 advanced
- **Interactive checkboxes** — tick off goals as you complete them, progress saved to localStorage
- **Progress bar** — live completion tracker per phase
- **20+ clickable resources** — direct links to the best free and paid tools
- **Dark / Light mode** — smooth toggle with preference saved across sessions
- **Slide-out navigation menu** — hamburger drawer with quick-jump links to every section
- **Kanji-style icons** — Japanese characters used as meaningful icon badges throughout
- **JLPT reference table** — N5 to N1 level guide with kanji count, vocabulary, and study hours
- **Fully responsive** — works on mobile, tablet, and desktop
- **Zen Kaku Gothic New font** — warm humanist typeface designed for both Latin and Japanese text

## 🗺️ The 4 Phases

| Phase | Level | Timeline | Focus |
|---|---|---|---|
| 01 Foundations | N5 | Month 1–3 | Hiragana, Katakana, 500 words, basic grammar |
| 02 Building Blocks | N4 | Month 4–9 | 1500 words, 300 kanji, speaking practice |
| 03 Immersion Era | N3 | Month 10–18 | Anime, manga, regular conversation |
| 04 Native Content | N2/N1 | Month 18+ | Novels, raw TV, JLPT certification prep |

---

## 🛠️ Built With

- **HTML5** — semantic structure
- **CSS3** — custom properties, CSS Grid, transitions, dark mode via `data-theme`
- **Vanilla JavaScript** — accordion, checkboxes, drawer menu, theme toggle, localStorage
- **Google Fonts** — Zen Kaku Gothic New · Shippori Mincho

**No frameworks. No build tools. No dependencies.**

## 🚀 Getting Started

### Option 1 — Open locally
Just download `index.html` and open it in any browser. That's it.

### Option 2 — Deploy on GitHub Pages
1. Fork or clone this repository
2. Go to **Settings → Pages**
3. Set branch to `main`, folder to `/root`
4. Save — your site will be live at `https://yourusername.github.io/repo-name`
